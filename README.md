# 🐦 Twitter Sentiment Analysis using Logistic Regression

## 📌 Overview

This project performs **sentiment analysis on tweets** using **Natural Language Processing (NLP)** and a **Logistic Regression model**.

It classifies tweets as:

* 😊 Positive (1)
* 😠 Negative (0)

---

## 🚀 Features

* Text preprocessing (cleaning, stemming)
* TF-IDF vectorization
* Logistic Regression model training
* Model evaluation on test data
* Handles large dataset (Sentiment140)

---

## 🧠 Tech Stack

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* NLTK

---

## 📂 Dataset

* Dataset used: **Sentiment140**
* Contains 1.6 million tweets labeled as positive or negative

👉 Note: Dataset is not included in this repo due to size.
You can download it from Kaggle:
https://www.kaggle.com/datasets/kazanova/sentiment140

---

## ⚙️ Project Workflow

1. Load dataset
2. Clean text (remove punctuation, stopwords)
3. Apply stemming
4. Convert text to numerical form using TF-IDF
5. Split data into training and testing sets
6. Train Logistic Regression model
7. Evaluate model accuracy
